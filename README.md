# Etch-a-Sketch using Turtle

Simple Etch-A-Sketch style renderer using Python's turtle and Pillow for image preprocessing.

Features
- Convert an input image to a black & white grid
- Render axis-aligned line art with Turtle

Requirements
- Python 3.8+
- Pillow

Quick start
1. Install requirements:
   pip install -r requirements.txt
2. Set `IMAGE_PATH` in `main.py` to your image file path or pass one in code.
3. Run:
   python main.py

Notes
- The script uses the `turtle` GUI; run it in a desktop session.
- If you don't have an image, change `IMAGE_PATH` to a valid PNG/JPEG file.
